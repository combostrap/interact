**Get Started**

```bash
# globally
npm install -g @combostrap/interact
# in a project
npm install @combostrap/interact
```

**Create a page**

```bash
mkdir -p src/pages
echo -e "# Hallo\nWorld" > src/pages/index.md
```

**Then**

```bash
interact start
```
